# Agente de IA com RAG completo
Esse projeto foi feito com o intuito de aprender coisas novas sobre agentes de IA e como cria-los, utilizando a biblioteca do langchain.<br><br>

## Como rodar
1. Copie o repositório localmente
2. Instale as dependências necessárias (abra o terminal *pip install -r requirements.txt*)
3. Rode um modelo localmente **OU** use uma API de algum modelo (utilizei um modelo local da Google pelo **lm studio**)
4. Rode cada bloco de código, de cima para baixo.
5. Pronto! Agora é só observar as respostas dadas pelo agente :)
